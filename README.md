# nodejs-data-processing-cli

## Description
**Data Processing Toolkit** — an interactive command-line application that performs various useful data processing operations. The tool works as a persistent Node.js process that accepts commands.


## Technical requirements
- Use 24.x.x version (24.10.0 or upper) of Node.js

## CLI Interface
- Install dependencies

```bash
npm install
```

- The program is started via npm-script `start`:

```bash
npm run start
```
